스킬들은 위에서 아래로 각각 Q -> W -> E 순이라고 보심됨다

자르반의 경우

 위에서 기준 1열 = Q // 2열 = W // 3열 = E

카직스의 경우
 
위에서 기준 1열 = Q // 2열 에 있는 스프라이트 딱 한개 = W ( 왜냐면 W의 스프라이트가 투사체 역할을 할 것이기 때문에 하나만 있듬 )  // 3열 = E스킬의 제자리 점프 준비 동작  //  4열 = E 스킬 포물선으로 점프 한후 착지 이펙트

세주아니의 경우가 조큼 귀찬음

위에서부터 기준 1열 = Q // 2열 = W // 3열 = 2열의 이펙트가 끝난후 얼음 정수의 위치에 생성되는 이펙트 
// 4열 = 3열 후반의 얼음지대 이펙트에 닿은 적에게 발생하는 스프라이트 ( 해당 적은 저 스프라이트 안에 갇힌것처럼 표현됨 )
